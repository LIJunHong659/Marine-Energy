# Changelog

## 0.10.0 - 2026-07-14

- 完成 Phase 9 / S8 固定容量多情景风险筛选；五个商业候选方案在六个统一的风况、登陆能力和客户合同情景中逐时回放，容量不随情景事后调整。
- 新增期望年值、最差20%概率质量均值、风险调整年值和情景后悔值，并检验容量选择对风险偏好权重的敏感性。
- 新增登陆能力替代计算；在300—700 MW登陆上限下，对照直接送电与固定综合方案的利用率、弃电和完整项目年值。
- 新增固定容量合同压力面；对7档算力服务价格和4档到岸氢价逐一比较五个方案，分别报告相对直接送电和完整项目两个盈亏门槛。
- 新增7、15、30和60日低风事件测试，区分商品型制氢方案与专用储氢—燃料电池保供方案，并计算按缺供价值回收固定成本所需的事件频率。
- 新增S8配置、候选清单、断点续算任务、结果图、独立审计脚本和风险筛选单元测试；74项测试全部通过，逐时最大功率平衡残差低于6e-14 MW。
- S8仍属合成情景下的离散候选筛选。情景权重未经历史统计估计，逐时调度具有完美预见，不得将筛选年值作为项目收益预测。

## 0.9.0 - 2026-07-14

- 完成 Phase 8 / S7 中国情境反事实价值评估；以绝对登陆点接纳上限替代仅按海缆容量比例缩放的大陆消纳假设，海缆增容不再自动增加登陆侧能力。
- 新增风电、海上光伏和波浪能的统一可再生能源入口；波浪能采用明确标注的滞后平滑筛选代理曲线，用于检验互补性而非替代同址后评估。
- 新增输氢管道小时能力、距离相关 CAPEX、固定运维和寿命，使售氢同时受需求、库存和管道交付能力约束。
- 新增完整项目年化筛选账：风电、光伏、波浪能、换流终端、海缆、登陆侧电网和岛上公共设施与灵活资产成本分账，避免把公共工程留在比较边界外。
- 新增中国情境证据登记、低中高成本表、全国利用率校准、局部登陆约束、海缆/电网/综合能源岛反事实、算力—氢价前沿、任务可移动性、资源组合及故障/低风压力案例。
- 新增已导出结果的独立审计脚本和48个固定随机时点审计表；8760小时代表性结果的所有检查通过，最大功率平衡残差低于 3e-14 MW。
- 所有结果仍以合成时序、筛选成本和机制性合同价格为基础；真实接入、气象、工程报价和客户合同到位前，不得视为项目可研结论。

- 完成三档大陆接纳约束与三档成本的九组年度规划、算力服务报价前沿、氢价前沿及联合商业情景。

- 新增连续90日低风投资规划，按关键负荷缺供价值内生选择电解槽、储氢和燃料电池容量。
