# 蓝海枢纽灵活资源的投资边界与长期保供价值

## 摘要

前一阶段的S5模型证明，在大陆接纳能力受限时，算力、制氢和储能能够显著减少弃能，并在海缆故障和长期低风期间提供替代通道。然而，固定容量下的运行增值尚未回答设备投资能否回收。S6由此将电池功率与能量、电解槽、储氢、燃料电池和算力配套光纤容量改为连续投资变量，在同一个逐时线性规划中决定建设规模和全年运行。各类设备按寿命和折现率计算资本回收，另计固定运维；PEM电解槽还计入栈体更换。风电、光伏和海缆在本轮视为各方案共有的既定条件，因此结果衡量灵活设施相对于直接输电的增量经济性。

8760小时合成案例给出了一组比S5更审慎的结果。大陆接纳中等受限、氢气交付价格22.5元/kg、算力服务报价360元/MWh-IT时，参照成本方案不建设任何灵活设施，直接保留125.73万MWh弃能。低成本方案配置76.84 MW电解槽和6.73吨储氢，增量净年值为1214万元。接纳约束进一步加重后，参照成本下51.27 MW电解槽进入方案，但增量净年值仅295万元，属于临界投资；高成本方案仍维持零投资。可见，接纳稀缺能够提高运行价值，却不能保证资本支出得到补偿。

合同价格前沿显示，在参照成本和中等接纳约束下，算力报价不高于1200元/MWh-IT时最优容量为零；报价达到1800元/MWh-IT时，300 MW-IT需求上限全部进入，增量净年值为10.89亿元。氢气价格由22.5元/kg提高至27.5元/kg后，57.48 MW电解槽开始进入，净年值约731万元；价格达到30元/kg后，容量稳定在72.63 MW，并配置5吨短周期库存。联合商业情景采用严重接纳约束、1800元/MWh-IT算力报价和35元/kg氢价，最优方案为300 MW-IT算力与51.13 MW电解槽，总初始投资187.36亿元，年化固定成本27.54亿元，增量运行边际39.50亿元，增量净年值11.96亿元。8个样本外合成年份的净年值均为正，P10为8.43亿元，最差年份为8.32亿元。

连续90日低风案例进一步说明了储氢的另一类价值。当氢气销售和算力需求关闭、基础负荷缺供价值取1万元/MWh时，模型选择6.11 MW电解槽、620.86吨储氢和7.31 MW燃料电池，将期望缺供电量由11,465.4 MWh降至84.1 MWh，增量净年值为6725万元。长期储氢投资由可靠性价值驱动，与平时销售氢气的商品价值应当分别论证。

以上数值仍建立在合成时序、完美预见和多项D级商业参数上。它们适合识别合同门槛、投资替代和保供价值，不能作为项目收益预测或融资结论。

## 一、从运行价值转向投资判断

S5的主要贡献在于恢复了电力供需错配对项目价值的影响。风电高峰与大陆需求低谷重合时，700 MW海缆即使处于完好状态，也可能受到登陆点接纳能力限制。算力可以通过光纤面向远端任务需求形成即时负荷，制氢则吸收超过快速通道能力的余电，并可经储氢实现更长周期的能量搬移。S5在固定设备规模下测得这些通道的运行增值，却没有让设备承担建设和年度维护成本。

这一差别会显著改变结论。供需错配情景中，S5的200 MW-IT算力、200 MW电解槽和配套储能都具有正运行价值；当S6要求每项容量回收初始投资时，基础报价下的参照成本方案选择零容量。弃能本身并不产生足够现金收入，只有客户合同、氢气销售、容量补偿或其他可兑现收益高于被放弃的送电价值和设备年化成本，投资才会发生。

S6保持S5的单一海上功率平衡。每小时的可再生出力只能用于关键负荷、海缆外送、电池充电、电解制氢、数据中心设施负荷或弃电。燃料电池回发和电池放电作为电力来源进入同一平衡。电力、氢气和算力收入仍以各自实际交付量确认，因而不存在同一份电量在不同出口重复计价的问题。

## 二、容量规划方法

### （一）连续容量变量

S6内生选择电池功率、电池能量、电解槽功率、储氢质量、燃料电池功率和算力容量。算力容量在本轮同时代表配套光纤服务能力。规划上界分别为300 MW、2400 MWh、400 MW、1500吨、100 MW和400 MW-IT。上界用于限定工程搜索范围，并不要求设备建至给定规模。

容量变量直接约束逐时运行。电池充放电不得超过功率容量，SOC上下限按能量容量的10%和90%计算；电解槽、燃料电池和算力服务分别受其容量约束；氢库存不得超过储氢容量。初末电池SOC和氢库存按容量比例闭合，避免通过消耗免费初始库存提高收益。海缆容量固定为700 MW，逐时送电上限仍由海缆物理可用率与大陆接纳系数共同决定。

### （二）投资年化

折现率为 \(r\)、寿命为 \(n\) 时，资本回收因子为：

\[
CRF=\frac{r(1+r)^n}{(1+r)^n-1}.
\]

年化固定成本等于初始投资乘以资本回收因子，再加年度固定运维。设备采用各自寿命，因而没有把不同资产粗略压缩到同一个年化比例。PEM栈体更换按初始投资的11%、每40,000满负荷小时一次，折成制氢电量的变动成本。目标函数最大化全年运行边际减去年化固定成本后的净年值。

这一目标适合灵活资产的增量筛选。风电、光伏和海缆是各方案共有条件，尚未纳入S6投资。完整项目的NPV还需要加入发电和输电投资、建设期、税费、债务、折旧和残值。

### （三）两层验证

规划层只使用全国可选算力池，以保持容量问题为线性规划。获得最优容量后，模型将其转换为固定系统配置，并交给S5逐时模型回放。回放层继续检查算力爬坡、光缆可用率、精确海缆损耗、海上功率平衡、电池SOC、氢库存和年末闭合。新增单元测试还要求零规划上界精确退化为S5直接输电，保证投资模型不会凭空创造价值。

## 三、成本与情景

电池参照成本以NREL 2025年报告的2024年功率分项372美元/kW和能量分项241美元/kWh为量级依据，按7.2元/美元工程汇率换算为2678元/kW和1735元/kWh。低成本情景按照2035年147美元/kWh的四小时系统低成本投影缩放，高成本情景加入海上工程溢价。NREL的成本口径来自美国陆上项目，只用于外部参照。

PEM电解槽采用美国能源部2024年记录中的1500、2000和2500美元/kW三档安装成本，折算后分别为1.08万、1.44万和1.80万元/kW。该记录同时提供30年寿命、5%固定运维、40,000小时栈体更换间隔和11%更换比例。储氢、燃料电池及算力配套光纤缺少适用于中国深远海项目的统一公开造价，采用宽区间筛选，并明确标记为D级。算力综合投资低、参照、高三档分别为2万、6万和12万元/kW-IT。

| 条件 | 大陆接纳设置 | 计算目的 |
|---|---|---|
| 宽松 | 全时段1.00 | 检验没有接纳稀缺时的投资 |
| 中等约束 | 低需求0.55，高风低需求0.32 | 延续S5主要供需错配案例 |
| 严重约束 | 低需求0.40，高风低需求0.20 | 检验更严重弃能能否支撑投资 |

三档成本与三档接纳条件形成九组年度规划。合同前沿在参照成本和中等接纳条件下分别改变算力报价与氢气交付价格。联合商业情景采用更高合同价格和严重接纳条件，用于观察算力与制氢能否同时进入。样本外分析固定联合情景的设计容量，生成8个带有自相关扰动的合成年份，并加入风况、电价、接纳能力、算力需求、48小时海缆或光缆故障以及一周低风事件。

## 四、成本与接纳约束的共同影响

| 接纳条件 | 成本情景 | 最优灵活资产 | 电解槽（MW） | 储氢（吨） | 增量净年值（百万元/年） |
|---|---|---|---:|---:|---:|
| 宽松 | 低、参照、高 | 无 | 0 | 0 | 0 |
| 中等约束 | 低 | 制氢 | 76.84 | 6.73 | 12.14 |
| 中等约束 | 参照、高 | 无 | 0 | 0 | 0 |
| 严重约束 | 低 | 制氢 | 88.02 | 37.11 | 52.21 |
| 严重约束 | 参照 | 制氢 | 51.27 | 0 | 2.95 |
| 严重约束 | 高 | 无 | 0 | 0 | 0 |

宽松条件下，直接输电已经能够消纳全部可再生电量，所有新增容量均为零。中等约束下，直接输电弃能125.73万MWh。低成本制氢方案减少33.37万MWh弃能，新增运行边际1.056亿元，年化固定成本9351万元，留下1214万元净年值。参照和高成本方案无法覆盖固定成本，弃能继续保留。

严重约束下，直接输电弃能177.40万MWh。低成本方案配置88.02 MW电解槽和37.11吨储氢，净年值增至5221万元。参照成本电解槽也会进入，但295万元净年值只占年化固定成本的3.3%左右。汇率、海上安装、可用率或氢价的轻微变化都可能改变该项选择，因此应将它视为临界状态。高成本方案仍为零容量，说明接纳约束加重并不能无限提高可承受造价。

电池在九组基础案例中均未进入。当前电价结构、85%往返效率、吞吐退化和外部参照造价使其日内套利与避弃电收益低于年化成本。后续算力报价达到2400元/MWh-IT后，电池才与高价值算力共同进入，用于将风电从低算力可用时段转移到高价值负荷。这一结果表明电池的价值取决于它所服务的下游产品，而不能只按减少弃电量评价。

## 五、合同价格门槛

### （一）算力服务

| 算力报价（元/MWh-IT） | 最优容量 | 增量净年值（百万元/年） | 剩余弃能（GWh） |
|---:|---:|---:|---:|
| 360 | 0 | 0 | 1257.28 |
| 900 | 0 | 0 | 1257.28 |
| 1200 | 0 | 0 | 1257.28 |
| 1800 | 300 MW-IT | 1089.32 | 86.14 |
| 2400 | 300 MW-IT，电池49.68 MW/364.76 MWh | 2613.93 | 32.05 |
| 3200 | 300 MW-IT，电池71.88 MW/565.16 MWh | 4681.36 | 13.14 |

算力进入区间位于1200至1800元/MWh-IT之间。由于价格步长较粗，本文不把1800元视为精确临界点。1800元情景中，300 MW-IT容量达到任务需求上限而未达到400 MW-IT规划上界，完成251.82万MWh-IT服务，减少117.11万MWh弃能。算力设施初始投资为180亿元，年化资本回收和固定运维合计26.63亿元，增量运行边际37.53亿元。

这一结果同时说明，算力的能源协同价值不足以承担完整设施投资时，项目需要真实的算力主营收入。若1800元报价没有客户合同、任务可迁移性和光纤SLA支持，模型中的正净年值无法兑现。算力报价应按训练、批量推理、科学计算、渲染等工作量分别校准，不宜使用一个统一价格长期替代市场调查。

### （二）氢气交付

氢气交付价格22.5元/kg时，参照成本电解槽为零；27.5元/kg时，57.48 MW电解槽开始进入，销售693.30万kg，增量净年值731万元。30元/kg时，电解槽增至72.63 MW，新增5吨库存用于短周期平滑，净年值2850万元。价格继续提高后，氢气需求上限成为主要约束，电解槽和储氢容量基本不变，净年值随价格增加。

制氢链因此存在两类商业状态。常规销售状态以电解槽和短时库存为主，储氢容量很小；长期保供状态需要数百吨库存和燃料电池，由可靠性收益支撑。两种状态不能用同一氢价和同一利用小时概括。

## 六、联合商业情景与样本外回放

联合情景下，最优容量为300 MW-IT算力和51.13 MW电解槽，未配置电池、储氢和燃料电池。算力吸收251.82万MWh-IT任务，制氢销售449.33万kg，剩余弃能15.06万MWh。灵活设施初始投资187.36亿元，其中算力占绝大部分；年度固定运维8.47亿元，资本回收19.07亿元，合计年化固定成本27.54亿元。相对直接输电，运行边际增加39.50亿元，净年值增加11.96亿元。

制氢与算力同时进入不意味着两项收益可以单独相加。算力需求在高报价下优先吸收大量余电，电解槽只利用剩余时段，最优功率由基础严重约束情景的51.27 MW略降至51.13 MW。S5测得的负交互项在投资层继续存在，表现为容量和利用率的相互挤压。

固定容量在8个样本外年份中的增量净年值平均为10.49亿元，中位数11.27亿元，P10为8.43亿元，最差8.32亿元，全部为正。平均减少弃能158.63万MWh。个别年份包含48小时海缆或光缆故障以及一周低风，结果仍保持正值，主要原因是1800元/MWh-IT算力合同为设备提供了较高主营收入。

这组样本外结果只能说明容量设计对所设扰动具有一定稳健性。8个合成年份数量有限，且每个年份的逐时调度仍拥有全年预见。真实项目需要历史再分析气象、预测误差、登陆点调度计划和工作量到达数据，才能报告具有统计含义的P10、最差值和价值在险。

## 七、储氢的长期保供价值

90日低风案例将风电可用率连续2160小时降至1%，关闭算力需求和氢气销售，使储氢价值只来自关键负荷保供。直接输电缺供11,465.4 MWh。

| 缺供价值（元/MWh） | 电解槽（MW） | 储氢（吨） | 燃料电池（MW） | 剩余缺供（MWh） | 增量净年值（百万元/年） |
|---:|---:|---:|---:|---:|---:|
| 5,000 | 5.80 | 588.96 | 6.10 | 668.97 | 11.00 |
| 10,000 | 6.11 | 620.86 | 7.31 | 84.10 | 67.25 |
| 20,000 | 6.15 | 624.81 | 7.75 | 12.09 | 181.60 |

缺供价值5000元/MWh时，模型保留669 MWh缺供，因为继续增加燃料电池和库存的年化成本高于剩余保供价值。提高到1万元/MWh后，储氢增至620.86吨，回发11,381.3 MWh，避免99.3%的缺供。2万元/MWh时，剩余缺供降至12.1 MWh。

参照情景下，长期保供设施初始投资约2.85亿元，年化固定成本3274万元。该数值显著低于联合算力情景的投资规模，因为岛上关键负荷约10 MW，所需回发功率有限；能量需求主要由储氢质量承担。若关键负荷扩大、低风时间延长或储氢成本上升，最优容量会相应变化。

缺供成本是可靠性价值代理。它需要通过容量合同、备用服务、自备电源替代成本、停产损失或保险安排才能转化为项目现金流。在没有付款机制时，避免缺供的社会价值不能直接计入股东收益。

## 八、对能源岛价值论证的启示

算力的核心优势在于任务需求与电力登陆点的地域解耦。只要工作量可迁移、数据与合规要求允许、光纤保持可用，海上风电高峰可以通过计算服务面向远端客户。S6进一步表明，这一优势需要足够高的服务合同价值。低报价只能证明算力运行时可利用低机会成本电力，无法支付服务器、供配电、冷却、海上结构和光纤建设。

氢能的价值随时间尺度分化。电解槽在余电时段生产并即时销售，依靠氢价和低机会成本电力回收投资；大规模库存与燃料电池负责跨周、跨季度保供，需要可靠性收益。两类收入可以共存，但合同依据、设备利用和风险来源不同，应分别测算。

电池承担小时级搬移。在基础合同价格下，参照造价使其保持零容量；当算力报价提高到2400元/MWh-IT以上，电池开始为高价值算力服务。这说明能源岛的多出口配置应围绕下游产品价值选择时间尺度，而不宜先设定电池、算力和氢能的固定比例。

项目价值可以沿三条现金来源展开。电力、氢气和算力形成商品收入；降低海缆和登陆点拥塞形成网络价值；跨季度储氢和通道冗余形成可靠性价值。商品收入可以进入常规购销合同，网络价值需要输电替代或系统服务安排，可靠性价值需要容量或保险机制。只有收入承担者、计量方式和结算规则明确后，模型中的技术价值才可转成财务价值。

## 九、局限与下一步

本轮完成了灵活资产的年化投资筛选，距离项目尽调仍有较大数据缺口。算力综合造价、储氢和燃料电池成本均为D级情景；海上数据中心服务器代际、冷却、UPS、海洋环境维护和光缆分摊尚未拆项。氢能链缺少压缩、管道、压力、最低负荷、启停、退化和安全间距。电池外部成本来自美国陆上系统，不能直接替代中国海上报价。

规划继续使用合成风况、价格和大陆接纳系数。样本外回放改变了外生年份，却仍采用完美预见调度。下一阶段应建立多情景容量规划，在历史风年、登陆点接纳计划和工作量样本上联合决定容量；运行层采用日前至日内滚动窗口，测量预测误差导致的弃能、违约和库存偏差。

完整财务模型需要纳入风电、光伏、海缆、输氢管道和登陆设施投资，并处理建设期、债务、税费、折旧、残值、设备更换和价格年份。届时可报告全项目NPV、IRR、偿债覆盖和风险分位数。碳属性、波浪能、设备故障率、海况可达性和秒级构网能力仍需分别建模，避免在逐时能量模型中用单一系数代替工程验证。

## 十、结论

S6把蓝海枢纽的讨论从运行价值上限推进到了灵活资产投资门槛。基础合同价格和参照造价下，模型可以选择保留弃能而不建设灵活设施；低成本、更严重接纳约束或更高商品价格会使电解槽和算力依次进入。算力服务报价的筛选进入区间位于1200至1800元/MWh-IT之间，氢气价格位于22.5至27.5元/kg之间。两者均依赖当前合成供需和成本口径，应通过实际合同继续收窄。

长期储氢在普通商业年度中没有进入联合方案，在90日低风和明确缺供价值下则成为主导容量。这一差异说明制氢销售与储氢保供需要两套价值证明。算力适合把高风低需求时段转化为即时服务收入，储氢适合处理超出其他通道能力的余电和跨季度可靠性，电池在高价值负荷出现后承担小时级搬移。多出口的合理配置由合同、成本和约束共同决定，无法预先用固定比例给出。

当前结果为项目提供了可复核的投资筛选方法，也给出了需要向电网、算力客户、氢气用户和设备供应商索取的数据清单。下一步工作的重点是用真实时序和报价替换合成输入，把灵活资产净年值进一步收敛为全项目现金流与风险区间。
