# Phase 0 assumptions

1. 合成时序只用于验证数据契约和回归行为，不代表广东实测风资源、市场电价或负荷。
2. 核心时间步长固定为 1 h；跨时区输入统一解析为 UTC 后检查连续性。
3. 电价允许为负，其余物理需求、负荷和碳强度必须非负。
4. 风电与输电可用率允许取 [0, 1] 内的连续值，以表达聚合场站的部分降额。
5. 电池往返效率 0.85 仅作为后续情景基准。若假定充放电效率对称，应各取 `sqrt(0.85)`，不能各取 0.85。
6. 制氢综合 SEC 基准 57.5 kWh/kg 已包含 stack 与 balance-of-plant。未来若改用分项法，必须关闭综合 SEC，防止辅助功耗重复计算。
7. PUE=1.15 及 1.08–1.25 区间是项目情景假设，不是对海上数据中心的已验证性能承诺。
8. 所有经济价格和成本在 Phase 0 均不用于结论；进入容量规划前还需统一币值年份、汇率、税口径和名义/实际折现率。

## Phase 1 additions

9. S0只允许风电供给关键负荷与海缆外送，不允许陆网反向购电。
10. 关键负荷具有物理优先级，剩余风功率才进入外送决策。
11. HVDC基准终端损耗3%、海缆额定负载损耗1.5%/100 km均为D级工程情景值。
12. 海缆损耗采用额定容量归一化二次代理；HVAC无功补偿、充电电流和热约束尚未进入S0。
13. `operating_margin`只扣除输电变动成本、弃电惩罚和失供惩罚，尚未扣除任何CAPEX、FOM、税费和融资成本。
14. 基准弃电惩罚为零；负电价下经济策略可主动弃电，强制外送策略则按可行上限送出。
15. 合成风资源、价格和故障窗口只用于调试和机制分析。

## Phase 2 additions

16. 电池为聚合式AC侧模型，未区分电芯、PCS、变压器和辅助系统的分项效率。
17. 85%往返效率对称拆分为充、放电效率；实际项目应使用厂商效率曲线校准。
18. SOC范围10%–90%、自放电和80 CNY/MWh吞吐退化成本均属于D级情景参数。
19. 全年度优化具有完美预见，故障前允许主动预充；突发故障能力另用固定初始SOC审计。
20. 固定储备约束只保留功率和能量裕度，不证明构网控制、RoCoF或最低频率合格。
21. 电池容量敏感性使用500 MW拥塞海缆，不能与700 MW基准价值直接混用。
22. 增量运行价值已经扣除吞吐退化成本，仍未扣除电池CAPEX、FOM、增容和更换成本。

## Phase 3 additions

23. 氢销售受每小时需求上限约束，储氢初末状态相同；未销售库存不产生收入。
24. 氢气运输、变动运维与淡化水成本为D级工程情景，海底输氢管道的压力、压缩、容量、损耗、故障与CAPEX尚未建模。

## Phase 4 / S3 additions

25. 算力主计量为MWh-IT。PUE定义为数据中心设施总能耗与IT设备能耗之比；1.08、1.15、1.25是情景值，不构成海上数据中心性能承诺。
26. 刚性与柔性算力到达量是合成的陆侧服务合同代理。刚性未完成量计入SLA惩罚，柔性任务必须在24小时最大延迟、500 MWh-IT队列上限和年度队列闭合内完成。
27. 海底光缆以MWh-IT/h服务交付能力和可用率表示，其目的在于表达服务履约链路；它不是光学带宽、通信时延或光缆电力损耗模型。
28. 算力服务价、变动运维、SLA惩罚、IT爬坡和光缆交付能力均为D级情景参数。Token、GPU小时、模型吞吐与网络流量仅在获得硬件及任务实测后作为后处理映射。

## Phase 5 / S4 additions

29. S4在同一时段共同优化电力外送、电池、制氢和算力设施耗电。电力、氢气和算力收益只按各自实际交付量计算，不能对同一电量重复记账。
30. S0—S4比较共享相同的风资源、关键负荷、海缆、价格与合同到达序列；各模式仅在启用的可选设备容量上不同。由于CAPEX和固定运维尚未纳入，这些结果只表示固定容量下的运行价值，不能宣称混合配置的投资最优性。
31. S4采用完美预见的年度调度。电缆和光缆故障案例用于检查能量与服务约束，不构成秒级暂态、通信时延或网络安全验证。

## Phase 6 / S5 additions

32. `grid_absorption_factor`是登陆点在每小时可接纳的海缆容量比例，与海缆设备可用率分开。0.32、0.55和1.00仅构成供需错配压力情景，尚未由真实省级调度数据校准。
33. 全国弹性算力需求是可选择接入的工作量池。基准需求200 MW-IT、服务价360 CNY/MWh-IT均为D级机制参数；未完成的弹性池需求不构成SLA违约。
34. S5仍以MWh-IT作为算力服务主计量。全国可调用并不意味着所有工作均可迁移；时延、数据主权、软件许可、输入数据体量、任务检查点和客户合约必须在实测工作量进入后逐类约束。
35. 氢低位热值取33.33 kWh/kg，燃料电池电效率取55%作为基准情景。燃料电池功率、耗氢和变动运维已进入模型，启停、最低负荷、退化、余热利用和设备CAPEX尚未进入。
36. 氢库存采用年度首尾相等。连续九十日低风场景允许模型在事件前制氢、事件中发电并在事件后恢复库存，因而代表具有年度预见的规划上限，不等同于真实滚动调度结果。
37. 八组合对照保持发电、价格、海缆和大陆接纳情景一致。组合之间出现负交互项表示资产争夺同一批余电，不能将各单项价值直接相加。
38. 等效海缆容量是在接纳系数随海缆额定容量同比放大的前提下计算的工程代理。若登陆点约束是绝对MW上限，扩建海缆不会按该数值改善接纳能力。
39. S5支持海上光伏容量进入共同母线，但代表性案例将其设为零，以保持与S0—S4可比。波浪能仍未进入配置schema和逐时发电方程。
40. S5全部经济结果仍未扣除CAPEX、固定运维、融资、税费和残值，只能解释为运行价值、成本上限或压力测试结果。

## Phase 7 / S6 additions

41. S6是灵活资产的增量规划。风电、光伏和海缆在所有方案中固定，未在S6重复计入投资；所得净年值不能替代全项目NPV。
42. 资本回收使用实际折现率和分设备寿命，未计建设期利息、债务比例、所得税、折旧、残值、保险和大修现金流。
43. 电解槽栈体更换按满负荷电量线性折算。真实退化还与启停、最低负荷、温度和动态工况有关。
44. 算力与光纤在容量规划中使用同一MW-IT容量和综合投资单价。GPU、CPU、存储、冷却、UPS、海上结构和光缆分摊尚未拆开，综合单价为D级筛选参数。
45. 三档成本中的美元换算采用7.2元/美元工程情景，不表示即期汇率。国外成本没有自动加入中国本地化、海上施工、认证、税费或供应链差异。
46. S6代表性规划只接收全国可选算力池，刚性和可延期合同仍由S5回放层处理。300 MW-IT任务需求是主情景上限，达到该值表示需求约束生效。
47. 样本外回放固定设计容量并扰动风况、电价、接纳能力、任务需求和通道故障。调度仍拥有每个样本年份的完美预见，因此尚未计入日前预测误差和滚动决策损失。
48. 八个样本外年份由确定性随机种子生成，只用于压力测试，不构成统计置信区间、价值在险估计或气候重分析。
49. 90日低风案例将关键负荷缺供成本作为保供价值代理。避免的缺供成本不是已经签署的容量收入，只有在可靠性合同、容量补偿或自备电源替代成本能够兑现时才可进入财务现金流。
50. 90日低风规划关闭氢气销售和算力需求，以隔离储氢回发的保供作用。该结果不含海上储罐安全间距、压力等级、压缩机、管道和消防系统的工程限制。
51. S6继续支持风电与光伏共同发电，但案例光伏为零；波浪能、潮流能和海洋温差能仍未进入发电方程。
