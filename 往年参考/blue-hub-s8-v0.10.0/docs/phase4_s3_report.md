# 蓝海枢纽 Phase 4 / S3 绿色算力与海底光缆服务调度报告

## 1. 模型边界

S3将绿色算力作为第三条价值输出路径，但保持与S0—S2相同的逐时风电、关键负荷和海缆边界。算力服务的主计量为MWh-IT，数据中心设施耗电由PUE换算：

\[
P_t^{DC}=PUE_tP_t^{IT}.
\]

PUE采用设施总能耗与IT设备能耗之比的通用定义。[U.S. DOE PUE metering guidance](https://www1.eere.energy.gov/manufacturing/datacenters/pdfs/task_force_metrics_recommendations.pdf) 当前模型用1.08、1.15、1.25作为乐观、基准、保守情景，均属于待工程校准的情景参数。

刚性任务在每小时到达，未完成部分进入SLA惩罚；柔性任务进入有限队列，必须在24小时内完成，年末队列必须为零。IT容量与爬坡均受约束。海底光缆表示为MWh-IT/h服务交付能力和可用率，限制已完成服务的大陆交付；它不属于电缆电力传输模型，也不产生电力损耗。

Token、GPU小时、推理吞吐和网络数据量未进入优化。它们必须使用具体硬件、模型和任务的实测能耗从MWh-IT结果后处理映射，当前不以假设系数制造算力收益。

## 2. 8760小时基准结果

基准采用100 MW-IT IT容量、100 MW-IT海底光缆服务能力、PUE=1.15和700 MW海底电缆。刚性任务为20 MWh-IT/h；柔性任务为8–12 MWh-IT/h的合成到达量。

| 指标 | 结果 |
|---|---:|
| S0电力直送运行收益 | 1,330.506 million CNY/年 |
| S3运行收益 | 1,430.278 million CNY/年 |
| 算力增量运行价值 | 99.772 million CNY/年 |
| 算力服务量 | 255,720 MWh-IT/年 |
| 总服务率 | 100% |
| IT容量因子 | 29.2% |
| 算力业务净运行边际 | 192.118 million CNY/年 |
| 最大柔性队列 | 236 MWh-IT |
| 最大队列状态残差 | \(2.842\times10^{-14}\) MWh-IT |

算力业务净运行边际与系统增量价值不同。前者只扣除算力变动运维和SLA成本；后者还反映为算力供电而减少的电力外送收入。这个分账避免把同一MWh同时计为电力收益和算力收益。

## 3. 场景测试

| 场景 | 算力增量运行价值 | 服务率 | 刚性未完成量 | PUE |
|---|---:|---:|---:|---:|
| 基准 | 99.77 million CNY | 100.0% | 0 MWh-IT | 1.15 |
| 算力价格降低30% | 40.60 million CNY | 100.0% | 0 MWh-IT | 1.15 |
| 算力价格提高30% | 158.94 million CNY | 100.0% | 0 MWh-IT | 1.15 |
| 算力需求提高150% | 239.48 million CNY | 100.0% | 0 MWh-IT | 1.15 |
| PUE=1.08 | 105.40 million CNY | 100.0% | 0 MWh-IT | 1.08 |
| PUE=1.25 | 91.74 million CNY | 100.0% | 0 MWh-IT | 1.25 |
| 24小时海缆故障 | 100.09 million CNY | 100.0% | 0 MWh-IT | 1.15 |
| 24小时光缆故障 | 94.75 million CNY | 99.81% | 480 MWh-IT | 1.15 |

海底电缆故障只切断电力外送，绿色算力仍可在海上风电支撑下完成并经光缆交付。海底光缆故障的影响则不同：24小时内算力服务输出归零，480 MWh-IT刚性任务不可补偿；柔性任务积压至队列，并在链路恢复后于时限内清偿。它区分了能源链路与算力服务链路两类故障。

## 4. 容量与价格边界

500 MW海缆拥塞场景下，50、100、150 MW-IT配置的增量运行价值分别为133.788、133.845、133.845 million CNY/年。50 MW-IT已能完成当前合成工作负荷，后续扩容几乎没有增益。这是工作负荷规模和时间形态导致的饱和，不能解释为真实项目只需50 MW-IT。

在本模型中，任务一旦进入合同到达序列，刚性任务必须服务或支付SLA成本，柔性任务必须在时限内完成。因此，改变服务价格主要改变运行边际和与电力外送的取舍，并不允许模型为了规避低价合同而删除任务。以基准合同结构计，服务价格降至300 CNY/MWh-IT时，S3运行收益低于S0，说明算力合同价格必须与海上供电机会成本一起评估。

## 5. 数值优化与边界

最初的最大延迟约束采用累计任务量不等式，8760小时问题会产生二次增长的矩阵非零元。模型现将其改写为逐时队列上界：队列只能包含最近最大延迟窗口内尚未完成的柔性任务。该式与累计服务条件等价，使8760小时基准求解时间约为2.4秒，同时保留队列状态和时限审计。

49项自动化测试通过，包括S0消融、PUE单调性、刚性SLA、柔性队列、最大延迟、光缆故障、IT容量、功率平衡和8760小时回归。S4将把电池、制氢储氢和算力放入同一个优化器，才可回答三种价值路径在同一小时究竟如何分配风电。
