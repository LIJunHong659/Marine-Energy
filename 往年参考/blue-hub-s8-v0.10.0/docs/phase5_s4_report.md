# 蓝海枢纽 Phase 5 / S4 完整联合调度与代表性案例报告

## 1. 联合模型

S4将电力外送、电池、制氢储氢和绿色算力放入同一个逐时线性规划。每小时的风电先保障关键负荷，余电再在海缆外送、电池充电、电解槽和数据中心设施耗电之间分配；电池放电、关键负荷失供和弃电共同闭合海上功率平衡。

氢业务按实际销售量记录收入，库存首尾闭合；算力以MWh-IT记录服务量，PUE换算数据中心设施耗电，刚性任务未完成量单独计入SLA成本，柔性任务受队列和最大延迟限制。PUE使用设施总能耗与IT设备能耗之比的通用定义。[U.S. DOE PUE metering guidance](https://www1.eere.energy.gov/manufacturing/datacenters/pdfs/task_force_metrics_recommendations.pdf)

该模型只比较固定容量下的运行调度。电池、电解槽、储氢、数据中心和光缆的CAPEX、固定运维、寿命与融资尚未进入目标函数，因此不能将S4收益高于单一模式解释为投资最优。

## 2. 8760小时联合基准

基准容量为700 MW海缆、100 MW/400 MWh电池、100 MW电解槽、72,000 kg储氢、100 MW-IT数据中心和100 MW-IT光缆服务能力。外生风资源、电价、氢需求和算力合同与S0—S3保持一致。

收入账本中，电力路径运行边际为1,132.095 million CNY，氢业务运行边际为240.112 million CNY，算力业务运行边际为192.118 million CNY；电池吞吐退化成本为18.752 million CNY。三项价值和成本均由各自的实际交付量导出，没有将同一MWh同时记入多个收入项。

## 3. 五方案公共边界对照

| 模式 | 年运行收益 |
|---|---:|
| S0 电力直送 | 1,330.506 million CNY |
| S1 电池 | 1,346.693 million CNY |
| S2 电池+氢能 | 1,446.048 million CNY |
| S3 绿色算力 | 1,430.278 million CNY |
| S4 联合配置 | 1,545.572 million CNY |

五种模式共享相同的风资源、海缆、电价、氢价和算力合同，因而可比较运行机制。S4相对S2多出99.524 million CNY/年的运行价值，反映在该合成合同条件下增加算力路径的边际收益。由于可选设备容量不同，且未计入任何投资成本，这不是五种投资方案的公平NPV排序。

## 4. 代表性案例

基准条件下，外送、电解槽和数据中心设施耗电分别为3.090 TWh、503.700 GWh和294.078 GWh。氢价提高30%或算力价格提高30%时，当前配置已经满足全部氢需求和算力合同，物理分配不再扩大，新增收益主要来自交付单价提高。

氢价降低30%时，售氢量由8.760 million kg降为4.443 million kg，电解槽用电由503.700 GWh降为255.500 GWh；氢能在制氢与算力两类非外送转换用电中的占比由63.1%降为46.5%。这说明氢路径在当前合成市场中具有可选商品属性，会主动让位给直接外送或既定的算力合同。

算力价格变化没有改变服务量，原因是刚性任务必须履约，柔性任务也必须在24小时内完成。价格变化改变的是合同经济边际和电力机会成本，而不会允许模型删除已接受的任务。若要研究算力合同是否应被接受，需要在下一阶段增加任务接入或拒绝决策。

24小时海缆故障时，外送下降并产生6.907 GWh弃电，但年度售氢和算力服务仍能保持100%。24小时光缆故障时，刚性算力未完成量为480 MWh-IT，柔性任务在恢复后清偿；年度氢销售保持不变。海缆与光缆复合故障进一步降低运行收益，说明能源出口和算力服务出口必须作为两个独立的可靠性链路评价。

## 5. 验证结论与下一步

53项自动化测试通过。S4在关闭算力时精确退化为S2，在关闭电池和制氢时精确退化为S3。电池SOC、氢库存、柔性队列、刚性任务、IT容量、PUE和两侧功率平衡均通过逐时审计。

下一阶段应加入容量投资年化、固定运维、碳属性和多目标约束，再进行容量网格、Pareto前沿与策略地图。届时可进一步加入算力合同接入决策、波动氢需求、光缆服务容量的真实通信口径和企业侧实测参数。
