# 蓝海枢纽 Phase 1 / S0 开发、验证与结果分析

## 1. 本轮实现

S0已经形成完整的固定容量运行模型。风电首先保障能源岛关键负荷，剩余功率在海缆容量和可用率范围内进行外送；送端功率经过终端线性损耗和海缆二次损耗后形成陆侧交付电量。模型逐时比较外送收入、输电变动成本和弃电惩罚，解析求得经济外送功率，并提供强制外送策略作为对照。

当前实现的核心价值在于为后续LP/MILP建立了独立的解析真值。将电池、制氢和算力接入后，任何综合优化结果都必须在关闭新增模块时退化回当前S0结果。

## 2. 基准结果

标准化系统为1000 MW风电、700 MW HVDC、100 km离岸距离、10 MW左右关键负荷。参数采用当前D级输电损耗情景。

| 指标 | 24 h | 8760 h |
|---|---:|---:|
| 可用风电 | 11,692.38 MWh | 3,994,434.78 MWh |
| 陆侧交付电量 | 10,986.66 MWh | 3,749,462.95 MWh |
| 输电损耗 | 465.73 MWh | 157,371.84 MWh |
| 送端损耗率 | 4.067% | 4.028% |
| 弃电率 | 0 | 0 |
| EENS | 0 | 0 |
| 运行收益 | 3.954 million CNY | 1,330.506 million CNY |
| 最大守恒残差 | \(4.62\times10^{-14}\) MW | \(5.68\times10^{-14}\) MW |

运行收益尚未包含投资、固定运维、融资与税费，因此不能解释为项目利润或NPV。

## 3. 海缆容量敏感性

在100 km、基准电价下：

| 海缆容量 | 弃电率 | 陆侧交付 | 运行收益 |
|---:|---:|---:|---:|
| 300 MW | 33.31% | 2.461 TWh | 0.919 billion CNY |
| 500 MW | 6.33% | 3.497 TWh | 1.264 billion CNY |
| 700 MW | 0 | 3.749 TWh | 1.331 billion CNY |
| 900 MW | 0 | 3.758 TWh | 1.334 billion CNY |
| 1000 MW | 0 | 3.762 TWh | 1.335 billion CNY |

500 MW增至700 MW带来的年运行收益增量约66.27 million CNY，相当于331,000 CNY/(MW·year)的新增容量年化成本上限。700 MW增至900 MW后，弃电已经消失，收益仅通过降低负载率和海缆损耗增加约3.04 million CNY，对应约15,200 CNY/(MW·year)。900 MW增至1000 MW的增量进一步降至约10,700 CNY/(MW·year)。

由此可以识别700 MW附近的物理容量拐点。经济最优仍需比较实际海缆与换流站的增量年化成本。在当前情景下，如果700至900 MW扩容的年化成本超过约3.04 million CNY，该扩容便不具备运行收益上的合理性。

## 4. 距离敏感性

700 MW海缆由50 km增加到200 km时，送端损耗率由3.51%升至5.06%，陆侧年交付量减少约60.25 GWh，年运行收益减少约20.55 million CNY。这一差值只反映输电损耗，尚未加入海缆长度增加带来的投资、施工和运维成本，因此真实距离惩罚会更大。

## 5. 电价敏感性

当前合成电价全部为正。电价倍率由0.7增至1.3时，外送功率和弃电率保持不变，运行收益近似线性变化。这说明单纯使用正数倍率无法识别路径切换阈值。后续引入制氢和算力后，应扫描电价绝对水平、负电价频率和持续时间，而不能只扫描倍率。

在构造的24小时−50 CNY/MWh事件中，经济策略停止外送并形成10.735 GWh经济弃电；强制外送策略继续送出相同的送端电量。经济策略相对强制外送减少运行损失约0.569 million CNY。这一结果验证了合同约束和弃电政策会直接改变调度边界。

## 6. 海缆故障

24小时海缆停运导致10.735 GWh网络弃电，陆侧年交付量减少10.303 GWh，年运行收益减少约3.61 million CNY，年度弃电率增加0.269个百分点。S0没有电池、制氢或柔性算力，因此所有剩余功率只能弃置。该结果将成为S1–S4衡量韧性增益的统一基准。

## 7. 验证与调试

- 27项单元及回归测试全部通过；
- 50组随机调度问题与每组20,001点稠密枚举交叉验证；
- 24 h解析手算、无风、负电价、强制外送、海缆停运和未来阶段容量误接入均有专项测试；
- 8760 h求解无缺失、无越界、无能量生成异常；
- ruff、mypy和compileall通过；
- 输出CSV和KPI均记录配置哈希与文件SHA-256。

## 8. 当前结论的边界

1. 700 MW只是当前合成资源下的物理拐点；
2. 输电损耗参数仍为D级情景值；
3. 模型尚未计入CAPEX、FOM、可用率概率分布和币值年份；
4. HVAC模型尚未包含无功和充电电流约束；
5. 风资源、价格和故障均非广东实测数据；
6. 当前不能比较电力、氢能和算力路径的经济优劣。

下一阶段应加入电池模块，并重点检验它在海缆拥塞、负电价和短时故障中的边际价值。S1仍不宜直接加入制氢和算力，以便先分离小时级储能价值。
